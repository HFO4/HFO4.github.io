2014-08-02
  • 正常的文章

原来作者个外链本来是自用的，后来心血来潮弄成了公用，结果小伙伴们太热情直接把流量用超了。这里放出基于七牛的树洞外链源码，内用使用说明，希望大家喜欢 :evil:

安装说明：
1、先导入“数据库文件”中的wailian.sql到你的数据库
如果导入出错，请执行SQL查询代替导入：

CREATE TABLE IF NOT EXISTSwailian(mingtext NOT NULL,key1text NOT NULL ) ENGINE=MyISAM DEFAULT CHARSET=utf8;

2、打开程序文件中的config.php并按提示修改即可使用。
对于文件类型限制，只在前台做限制,如需更严格限制，请参阅七牛官方文档上传策略（PutPolicy）http://developer.qiniu.com/docs/v6/api/reference/security/put-policy.html。
更改LOGO时最好不要改变大小，如果改变大小，请修改容纳logo的table的长宽以防止错位。
请务必在页面底部加上本站链接：http;//aoaoao.me
本程序采用七牛云存储，由于七牛名声不太好，最近已开发出基于又拍云的外链，正在公测中。
