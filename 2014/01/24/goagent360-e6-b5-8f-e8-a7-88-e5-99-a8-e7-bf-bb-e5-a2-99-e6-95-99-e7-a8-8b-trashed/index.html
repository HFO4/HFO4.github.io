GoAgent+360浏览器翻墙教程

2014-01-24
  • 正常的文章

GoAgent是一个基于Google Appengine的，全面兼容IE，FireFox，chrome的代理工具，使用Python和Google App EngineSDK编写，程序可以在MicrosoftWindows，Mac，Linux，Android，iPod Touch，iPhone，iPad，webOS，OpenWrt，Maemo上使用。GoAgent出口地址使用的是美国加利福尼亚州山景城Google数据中心IP段。
下面为大家介绍一下如何用GoAgent呼吸“墙外的空气”

①首先确定你是否有Google账号，如果没有，可以注册。

②注册之后请登录。

[

③单击“Create Application”

[

③填写“Application Identifier:”，点击右边的按钮检查是否可用。随便填写“Application Title”

[

④拉到最下面，同意协议后点击“Create Application”

[

[

⑤下载goagent：

⑥解压后打开goagent文件夹下server/uploader.bat，输入刚才填写的“Application Identifier”，回车

[

⑦填写邮箱，填完后回车

[

⑧填写Google账户的密码，填完回车

[

等一会后，如果显示下面内容，表示成功

[

⑨打开goagent/local/proxy.ini，在画红圈处填写你的APPID（就是刚才的Application Identifier）填完后保存

[

打开\goagent\local\goagent.exe

[

接下来配置浏览器。（本次以chrome内核的360浏览器示范，其他chrome内核的过程也差不多）

首先安装Proxy SwitchySharp插件。

[

打开Proxy SwitchySharp选项，点击“从文件恢复”

[

选择下载好的local目录下的SwitchyOptions.bak文件，确认覆盖

[

打开浏览器设置，导入证书

[

点击“导入” 然后点击“下一步”

[

选择local/CA.crt文件点击“下一步”

[

导入成功[

选择浏览器插件栏中的灰色地球

[

选择“GoAgent”，灰色地球变蓝，翻墙成功！

[

科普一下GoAgent和GoAgent PAC的区别：GoAgent是所有网站都走GAE流量（速度较慢，但是所有网站都能上。）GoAgent PAC是根据pac文件检索地址需要使用GAE时才会使用（速度较快，但是有些网站会上不去）我还是推荐PAC，PAC上不去时再用GoAgent(省GAE流量，速度也快！）

要注意的是翻墙浏览网页过程中goagent不能退出

[
