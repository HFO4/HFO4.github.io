给大家推荐一个功能强大、界面漂亮的取色器。
GitHub：https://github.com/bgrins/spectrum
Demo和一些说明：http://bgrins.github.io/spectrum/
下面是一个取色器的预览：
[


可以看出 功能很强大，预设颜色、滑动取色、透明度什么的都有了。首先下载CSS和javascript文件，然后在页面中调用

<script src='spectrum.js'></script>
<link rel='stylesheet' href='spectrum.css' />
然后，来创建一个基础功能的取色器。基础取色器是建立在文本框之上，取到的颜色会以16进制的形式输入进文本框，基础取色器只有简单的滑动取色功能。首先给对象文本框设置一个id： 
<input type='text' id="custom" />
然后是js代码，其中的color值为取色器默认值。 
<script>
$("#custom").spectrum({
    color: "#f00"
});
</script>
来体验一下效果吧： 如果想在取色器中设置透明度，可以看看这个例子： 
$("#showAlpha").spectrum({
    showAlpha: true
});
看下效果： 如果想禁用取色器，js中加上“disabled: true”就行了 取色器中也可以添加预选色，看下面的例子： 
$("#showPalette").spectrum({
    showPalette: true,
    palette: [
        ['black', 'white', 'blanchedalmond'],
        ['rgb(255, 128, 0);', 'hsv 100 70 50', 'lightyellow']
    ]
});
我们可以看出来，设定预选色支持RGB、HSV等模式，看demo： 如果需要，我们也可以只显示预选色： 
$("#showPaletteOnly").spectrum({
    showPaletteOnly: true,
    showPalette:true,
    color: 'blanchedalmond',
    palette: [
        ['black', 'white', 'blanchedalmond',
        'rgb(255, 128, 0);', 'hsv 100 70 50'],
        ['red', 'yellow', 'green', 'blue', 'violet']
    ]
});

先要去掉“choose”按钮，可以通过“showButtons”实现：

$("#hideButtons").spectrum({
    showButtons: false
});
