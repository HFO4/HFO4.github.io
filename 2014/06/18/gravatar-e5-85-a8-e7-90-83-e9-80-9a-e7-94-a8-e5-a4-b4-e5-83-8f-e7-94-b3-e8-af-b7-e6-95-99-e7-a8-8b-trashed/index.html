Gravatar 全球通用头像申请教程

2014-06-18
  • 正常的文章

在逛博客时我们经常发现别人留言时有花里胡哨的头像，但自己留言出现的却是难看的默认头像，更神奇的是，这些博客没有头像功能，拿这些头像是哪来的？更更神奇的是，博客系统好像认识这些人似得，他们在不同博客留言都会显示自己的头像！这些神奇东来源于Gravatar
Gravatar是国外的一个头像服务，用户可以在其网站上上传自己的头像，上传前，Gravatar要求用户留下邮箱。上传成功后，只要用户在支持Gravatar的博客上留言并留下自己邮箱时，博客就会自动从Gravatar获取头像并显示？是不是很神奇？是不是申请？不幸的告诉你，Gravatar被墙了，本文就结束了

 

 

 

 

 

 

 

 

 

吗？没有！我们有GoAgent啊！不幸的告诉你，Gravatar已禁止GoAgent登录其网站。所以，本文真的真的就结束了

 

 

 

 

 

 

 

 

吗？没有。我们还有VPN啊！上一篇文章的Tenacy就是个不错的选择：传送门

连接好后，我们打开Gravatar的主页http://en.gravatar.com/

点右上角的“Sign in”
[


如果你有WordPress的账号，就直接登录，如果没有，点击”Need an account?”注册一个
[
从上到下依次填写邮箱、昵称、密码，“Sign up”
[
这样，账号就创建成功了，还要到邮箱里验证一下
[
点“Activate Account”激活账号
[
好！账号已激活，点“Sign in to Gravatar”[
现在就可以添加一个头像了，点“Add one by checking here”
[
这里提供了4种上传方式，第一种最实用也最常见，从电脑中上传。
[
选择好图片后点“Next”
[
拖动、缩放矩形边框对头像进行简单剪裁
[
这里可以预览一下头像，如果没问题，就点“Crop and Finish”
[
好了，到了关键一步了，这一步不要乱点，是给你的头像分级，一般都是第一个G级，也就是全龄级别，一般的网站只显示这一级别，过滤其他级别，所以不要选别的，就选第一个。
[
这样，头像就上传成功了！断开VPN我们来测试一下。
[
找到一个支持Gravatar的博客悄悄告诉你，树洞就支持哦！
找到一篇文章，来到评论栏，输入你的评论、注册Gravatar时填的邮箱，发表评论看看。
[
哈哈，头像已经显示出来了！
[
