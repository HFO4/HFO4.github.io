让C7V5支持使用UPYUN API获取题图主题色

默认情况下C7V5只支持从七牛获取主题色，如果博客的图片不放在七牛的话就没法获取。上次被七牛坑了之后博客图片全部转移到了又拍云，而主题色获取也只要修改一点JS代码即可：
打开主题目录下js/script.js，大概在第269行（已格式化）：

if (T) d(T.split(","));
else if (C7V5.qiniu_host &amp;&amp; V.indexOf(C7V5.qiniu_host) &gt; -1) e.getJSON(V.split("?")[0] + "?imageAve",
function(e) {
        var t = c(e.RGB);
        d(t),
        z &amp;&amp; localStorage.setItem("pcc_" + V, t.join(","))
});
`</pre>
修改为：
<pre class="language-javascript">`
if (T) d(T.split(","));
else if (C7V5.qiniu_host &amp;&amp; V.indexOf(C7V5.qiniu_host) &gt; -1) e.getJSON(V.split("?")[0] + "!/excolor/1/exformat/hex",
function(e) {
        var t = c(e[0].color);
        d(t),
        z &amp;&amp; localStorage.setItem("pcc_" + V, t.join(","))
});
`</pre>

### 评论框添加插入图片按钮

主题commentform.php大概第117行：
<pre class="language-php">`
&lt;?php if ( c7v5_get_option('smilies') &amp;&amp; function_exists('c7v5_smilies') ) : ?&gt; 
       &lt;i class="icon-smilies" title="表情"&gt;&lt;/i&gt;
&lt;div class="comment-smilies"&gt;&lt;?php c7v5_smilies(); ?&gt;&lt;/div&gt; `</pre>
在其中增加一个插入图片的按钮：
<pre class="language-php">`
&lt;?php if ( c7v5_get_option('smilies') &amp;&amp; function_exists('c7v5_smilies') ) : ?&gt; 
       &lt;i class="icon-smilies" title="表情"&gt;&lt;/i&gt;
       &lt;i class="icon-pic" title="图片" onclick="addimg();"&gt;&amp;#xE3C4;&lt;/i&gt;
&lt;div class="comment-smilies"&gt;&lt;?php c7v5_smilies(); ?&gt;&lt;/div&gt; `</pre>
主题选项-自定义代码中添加：
<pre class="language-javascript">`
function addimg(){

function auto_comment_image( $comment ) {// by https://mufeng.me
      global $allowedtags;
      $content = $comment["comment_content"];
      // alt部分自行填写
      $content = preg_replace('/((https|http|ftp):\/\/){1}.+?.(jpg|gif|bmp|bnp|png)$/is','&lt;img src="$0" alt="" /&gt;',$content);
      //允许发布img标签
      $allowedtags['img'] = array('src' =&gt; array (), 'alt' =&gt; array (),'class' =&gt;array());
      // 重新给$comment赋值
      $comment["comment_content"] = $content;
      return $comment;
}
`</pre>

### 给LOGO和标题添加阴影

最近把首页大图换成了我老婆，但默认白色的logo看不清了，所以给logo加上了阴影特效（前提是你的logo是svg绘制的）\
主题设置-自定义CSS中添加：
<pre class="language-css">`
#logo {
        -webkit-filter: drop-shadow(0px 0px 5px black);
}
#site-description,#site-title{
        text-shadow:0px 0px 5px #000 ;
}
`</pre>
